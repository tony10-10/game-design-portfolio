## 📂 项目一：守望先锋 5v5 机制下坦克站位趋势演变与核心价值重构

**The Trend of Tank Positioning Changes and the Core Value Redefinition Under the 5v5 Mechanism of Overwatch**

从多赛季单排宗师坦克的第一手视角出发，论证守望先锋从 6v6 到 5v5 的机制切换如何重构了坦克角色的设计价值——核心命题是：**5v5 让坦克从"空间资产"变成了"时间资产"，它控制的不再是地图区域，而是整局对局的节奏主动权。**

**研究亮点：**
- 提出"用可再生资源换取空间"的底层资源模型，并以此贯穿全文论证
- 原创"机动力三层分级框架"（高机动 / 线性机动 / 锚点型），系统化解释坦克站位逻辑
- 拆解所有坦克大招的控场设计共识，论证坦克大招的积累劣势
- 以 2026 OWCS Tokyo Champions Clash 决赛实战数据加以印证
- 升维至游戏策划视角，提出平衡困局的结构性本质与原创英雄设计思路

## 📂 项目二：卡牌构筑与 Roguelike 成长系统设计横向研究

以《炉石传说：酒馆战棋》《The Bazaar》《金铲铲之战》三款游戏为样本，横向拆解其核心成长机制、资源运营逻辑与随机性设计哲学。

**核心命题：掌控感与随机性的设计张力**——随机性是这类游戏重复可玩性的生命线，但过高的随机影响因子会稀释玩家掌控感。三款游戏恰好代表了这个张力谱系上三个截然不同的解法。

**研究亮点：**
- 建立"随机影响因子 × 运营决策权重"的设计坐标系
- 以数学模型量化三款游戏的胜率构成差异
- 系统对比经济爆发形式、Meta 趋同程度、连败补偿机制等十余个维度
- 提炼出对系统策划的核心设计启示
